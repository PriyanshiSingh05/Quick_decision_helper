#  Quick Decision Helper

A lightweight web tool designed to reduce decision fatigue by helping users quickly choose the best option based on simple criteria.

---

##  Problem

In everyday life, small decisions like:

* What to eat 
* What task to prioritize 
* Where to go 

often take disproportionate time due to too many options and no structured way to evaluate them.

This leads to **overthinking, delays, and reduced productivity**.

---

##  Solution

Quick Decision Helper provides a simple interface where users can:

* Input multiple options
* Evaluate them based on:

  * Cost 
  * Time 
  * Personal Preference 
* Instantly get the **best option**

---

##  How It Works

The system uses a **weighted scoring model**:

* Lower cost → better
* Lower time → better
* Higher preference → better

It also **normalizes real-world inputs** (like ₹250 or 30 minutes) into a comparable scale.

Final score is calculated using:

```
Score = 0.3 * (Cost Score) + 0.3 * (Time Score) + 0.4 * (Preference)
```
